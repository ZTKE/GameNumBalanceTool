## 1. 权力核心 (Who Rules? How to Change?)

### Victoria 3 中的日本权力结构

#### 初始政体设定 (1836年)
| 维度 | 状态 | 说明 |
|------|------|------|
| **治理原则** | 君主制 (Monarchy) | 幕府将军统治，天皇名义 |
| **权力分配** | 独裁 (Autocracy) | 无选举制度 |
| **继承机制** | 幕府继承 | 德川家族世袭 |
| **国家宗教** | 国家宗教 | 神道教+佛教混合 |
| **社会等级** | 武士制度 | 四民等级制 |

#### 核心利益集团 (Interest Groups)

| IG名称 | 初始权重 | 意识形态 | 政治立场 |
|--------|----------|----------|----------|
| **幕府 (Shogunate)** | 极高 | 传统主义、保守 | 德川政权核心 |
| **武士 (Samurai/Landowners)** | 高 | 传统主义、军事 | 维护武士特权 |
| **天皇派 (Imperialists)** | 低→高增长 | 皇权至上 | 推动天皇复权 |
| **实业家 (Industrialists)** | 极低→高增长 | 现代化、工业化 | 明治维新推动者 |
| **知识分子 (Intelligentsia)** | 低→高 | 现代化、学习西方 | 西学东渐 |
| **军队 (Armed Forces)** | 中→极高 | 军国主义（后期） | 军部崛起 |
| **教会 (Devout)** | 中 | 神道教保守 | 维护传统信仰 |
| **农民 (Peasants)** | 大量 | 多样化 | 农村基础 |

```
    │           │                   │
    │           │                   └→ [明治维新] → 天皇复权
    │           │                           │
    │           │                           ├──→ [宪政建立] → 议会制
    │           │                           │
    │           │                           └→ [军部崛起] → 军国主义
    │           │
    │           └→ [现代化改革] → 工业化
```

## 2. 政治工具箱 (All Laws, Policies, Abilities)

### Victoria 3 日本法案矩阵

#### 权力结构法案
```
治理原则选项：
  ✓ 君主制（幕府/天皇）
  ✓ 议会制（明治后）
  ✓ 军部独裁（二战路线）

权力分配演进：
  独裁 → 地产投票 → 户主投票 → 普选权（战后）
  军部独裁（二战）

继承机制：
  幕府继承 → 天皇世袭 → 议会选举（战后）
```

#### 日本特色法案路径
| 法案类型 | 日本演进路径 | 关键历史节点 |
|----------|--------------|--------------|
| **治理原则** | 幕府→天皇→宪政→军部 | 明治维新→二战 |
| **军事体制** | 武士→国民军队→军部主导 | 军事国家化→军国主义 |
| **经济体制** | 传统→农本→国家导向→战争经济 | 国家资本主义 |

## 3. 社会博弈 (Domestic Power Dynamics)

### Victoria 3 日本内部势力互动

#### 利益集团博弈图谱
```
      [西方压力/开放国门]
              ↓
┌─────────────────────────────┐
│                             │
幕府 ←──冲突──→ 倒幕联盟 ←──冲突──→ 天皇派
(保守)      (萨摩长州)        (皇权复归)
    │              │                 │
    └─────戊辰战争──────┘
           │
           ↓
       [明治维新]
           │
           ↓
    ┌────┴────┐
    │         │
实业家崛起    军部崛起
(现代化)     (军国主义)
```

#### 日本特有的政治博弈
1. **幕府 vs 天皇派**: 倒幕运动的核心冲突
2. **武士阶层废除**: 传统武士 vs 现代化改革
3. **宪政 vs 军部**: 议会民主 vs 军国主义（二战前）
4. **财阀 vs 工人**: 工业资本 vs 劳工运动

#### 戊辰战争触发条件
| 条件 | 触发阈值 | 结果 |
|------|----------|------|
| 西方干预压力 | 开放国门事件 | 幕府威信下降 |
| 倒幕联盟形成 | 萨摩长州联合 | 内战准备 |
| 武士激进度 | 武士不满 > 80% | 内战爆发 |

## 6. 开发者评论 (Design Philosophy Analysis)

### Victoria 3 日本设计解读

**设计意图**: 日本展示了"被迫开放→主动现代化→军国主义扩张"的三阶段路径。开发者意图模拟：
1. **幕府崩溃**: 西方压力如何触发内部革命
2. **明治维新**: 快速现代化的历史奇迹
3. **军部崛起**: 现代化军队如何走向军国主义

**玩家体验**: 日本玩家的核心体验是：
- 倒幕运动的战略选择（幕府路线 vs 天皇路线）
- 明治维新的现代化速度
- 二战前的宪政 vs 军部选择

## 1. 权力核心 (Who Rules? How to Change?)

### Victoria 3 中的法国权力结构

#### 初始政体设定 (1836年)
| 维度 | 状态 | 说明 |
|------|------|------|
| **治理原则** | 君主制 (Monarchy) | 七月王朝，路易-菲利普一世 |
| **权力分配** | 地产投票 (Landed Voting) | 有限的选举权 |
| **国家宗教** | 国家宗教 (State Religion) | 天主教为国教 |
| **选举周期** | 有议会 | 但王权主导 |

#### 法国的特殊政治地位
法国在 Victoria 3 中是最具"革命传统"的国家：
- 1789年大革命的历史遗产
- 1830年七月革命（刚发生）
- 1848年二月革命（即将发生）
- 多次共和与君主制的反复

#### 核心利益集团 (Interest Groups)

| IG名称 | 初始权重 | 意识形态 | 政治立场 |
|--------|----------|----------|----------|
| **奥尔良派 (Orléanists)** | 高 | 温和君主主义 | 支持七月王朝 |
| **正统派 (Legitimists)** | 中 | 保守君主主义 | 支持波旁复辟 |
| **共和派 (Republicans)** | 低→高增长 | 民主共和主义 | 推动共和制 |
| **波拿巴派 (Bonapartists)** | 低→中增长 | 威权民族主义 | 推动帝国复辟 |
| **天主教会 (Devout)** | 高 | 保守天主教 | 维护宗教地位 |
| **地主阶级 (Landowners)** | 高 | 传统主义 | 维护贵族利益 |
| **实业家 (Industrialists)** | 中→高增长 | 自由主义 | 推动经济改革 |
| **知识分子 (Intelligentsia)** | 中 | 自由主义、共和主义 | 政治改革核心 |
| **工会 (Trade Unions)** | 极低→后期增长 | 社会主义 | 工人权益 |
| **农民 (Peasants)** | 大量 | 多样化 | 农村保守势力 |

#### 权力更替的多条路径
```
七月王朝(1830-1848)
    │
    ├──→ [1848革命] → 第二共和国(1848-1852)
    │           │
    │           ├──→ [波拿巴政变] → 第二帝国(1852-1870)
    │           │                   │
    │           │                   └→ [1870战败] → 第三共和国
    │           │
    │           └→ [激进共和] → 社会主义倾向
    │
    └→ [正统派复辟] → 波旁王朝 (历史失败路径)
```

## 2. 政治工具箱 (All Laws, Policies, Abilities)

### Victoria 3 法国法案矩阵

#### 权力结构法案（法国特有多选项）
```
治理原则选项：
  ✓ 君主制 (七月王朝初始)
  ✓ 共和制 (革命后)
  ✓ 总统制 (第二共和国)
  ✓ 皇帝制 (第二帝国)

权力分配演进：
  地产投票 → 户主投票 → 财富投票 → 普选权
  （法国有最快的普选权改革速度）

继承机制：
  君主制 → 选举制（共和）
  波拿巴继承 → 威权政变
```

#### 法国特色法案路径
| 法案类型 | 法国特殊路径 | 关键触发条件 |
|----------|--------------|--------------|
| **治理原则** | 君主→共和→帝国→共和（反复） | 1848事件链 |
| **投票制度** | 地产→普选（快速） | 共和派推动 |
| **公民权利** | 无公民权→文化排斥→多元文化 | 自由主义改革 |
| **宗教政策** | 国家宗教→宗教自由→完全分离 | 知识分子推动 |
| **经济体制** | 传统→农本→自由放任→干预 | 实业家主导 |

#### 关键历史法案
| 法案 | 历史背景 | 游戏效果 |
|------|----------|----------|
| **拿破仑法典** | 民法典统一 | 法治加成 |
| **世俗化法案** | 教会与国家分离 | 知识分子支持 |
| **劳工保护法** | 1848后社会改革 | 工会支持 |

```
        [共和革命压力]
              ↓

│                             │
正统派 ←──冲突──→ 奥尔良派 ←──冲突──→ 共和派
(波旁)       (七月王朝)        (民主)
│              │                 │
└──────┬───────┴─────────────────┘
       │
       ↓
   [波拿巴派崛起]
       │
       ↓
   [第二帝国]
```

#### 法国特有的政治派系斗争
1. **三派君主之争**: 正统派(波旁) vs 奥尔良派 vs 波拿巴派
2. **共和派激进运动**: 1848年街头革命 → 第二共和国
3. **教会与世俗化冲突**: Devout vs Intelligentsia
4. **巴黎公社**: 工会+社会主义者的激进革命（1871）

#### 1848革命事件链
| 步骤 | 事件 | 游戏机制 |
|------|------|----------|
| 1 | 经济危机触发 | 银行崩溃事件 |
| 2 | 巴黎街头抗议 | Pop激进度飙升 |
| 3 | 七月王朝崩溃 | 君主制废除 |
| 4 | 第二共和国建立 | 共和制法案 |
| 5 | 波拿巴当选总统 | 波拿巴派崛起 |
| 6 | 政变或合法改革 | 多条路径选择 |

#### 人口阶层 (Pop) 政治诉求
| Pop类型 | 主要诉求 | 激进度来源 |
|---------|----------|------------|
| **贵族** | 维护君主制 | 共和威胁 |
| **工人** | 工作条件、社会主义 | 经济危机 |
| **农民** | 土地、保守稳定 | 激进改革威胁 |
| **知识分子** | 共和、自由改革 | 君主制阻碍 |

## 6. 开发者评论 (Design Philosophy Analysis)

### Victoria 3 法国设计解读

**设计意图**: 法国被设计为"政治实验室"，展示19世纪最复杂的政体反复。开发者意图模拟：
1. **革命传统**: 法国是唯一有多次革命事件链的国家
2. **派系竞争**: 三大君主派系+共和派的多维博弈
3. **历史敏感性**: 每个政治事件都有历史对应

**玩家体验**: 法国玩家面临的核心挑战是"如何管理革命周期"。可以选择：
- 维护七月王朝（稳定路线）
- 引导共和革命（改革路线）
- 波拿巴帝国（威权路线）

**机制创新**:
- 多派系君主竞争机制
- 事件链驱动的政体变更
- 巴黎公社的特殊内战

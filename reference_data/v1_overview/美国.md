## 1. 权力核心 (Who Rules? How to Change?)

### Victoria 3 中的美国权力结构

#### 初始政体设定 (1836年)
| 维度 | 状态 | 说明 |
|------|------|------|
| **治理原则** | 共和制 (Republic) | 总统制，无君主 |
| **权力分配** | 总统独裁 → 逐步民主化 | 早期总统权力集中 |
| **投票制度** | 地产投票 (Landed Voting) | 仅限土地拥有者 |
| **选举周期** | 4年总统任期 | 可修改选举周期 |

#### 核心利益集团 (Interest Groups)

| IG名称 | 初始权重 | 意识形态 | 政治立场 |
|--------|----------|----------|----------|
| **民主党 (Democratic Party)** | 高 | 传统主义、奴隶制支持者 | 保守南方势力 |
| **共和党前身 (Whigs)** | 中高 | 自由主义、工业保护 | 北方工业势力 |
| **地主阶级 (Landowners)** | 高 | 传统主义、保守 | 南方种植园主 |
| **实业家 (Industrialists)** | 中→高增长 | 自由市场、工业化 | 北方工厂主 |
| **知识分子 (Intelligentsia)** | 低→高增长 | 废奴主义、改革 | 废奴运动核心 |
| **工会 (Trade Unions)** | 极低→后期增长 | 工人权益 | 工业化后出现 |
| **教会 (Devout)** | 中 | 多样化 | 宗教道德观 |

#### 权力更替机制
1. **选举改革路径**: 地产投票 → 户主投票 → 地产+财富投票 → 普选权
2. **废奴引发的权力重组**: 削弱地主阶级 → 南方分裂 → 内战重构
3. **工业资本崛起**: 实业家权重增长 → 取代地主成为主导力量

## 2. 政治工具箱 (All Laws, Policies, Abilities)

### Victoria 3 法案矩阵

#### 权力结构法案
```
治理原则: 共和制 ✓ | 总统制 ✓ | 议会制 ✗
权力分配: 省独裁 → 总统独裁 → 地产投票 → 普选权 (渐进)
继承机制: 无（选举制）
领袖特质: 随机生成，可通过事件获得历史人物特质
```

#### 经济制度法案
| 法案类型 | 美国可选路径 | 关键利益集团支持 |
|----------|--------------|------------------|
| **贸易政策** | 保护主义 → 自由贸易 | 实业家偏好自由贸易，地主偏好保护 |
| **经济体制** | 传统主义 → 农本经济 → 自由放任 → 干预主义 | 实业家推动自由放任 |
| **税收制度** | 人头税 → 土地税 → 消费税 → 所得税 | 政治改革推动 |
| **劳动权利** | 无童工限制 → 童工禁止 → 工作时间限制 | 工会推动 |
| **社会福利** | 无社会保障 → 社会保险 → 养老金 | 知识分子+工会推动 |

#### 社会制度法案
| 法案类型 | 美国典型演进路径 |
|----------|------------------|
| **奴隶制度** | 奴隶合法 → 内战废除 → 无奴隶 |
| **移民政策** | 无限制 → 文化排斥 → 多元文化主义 |
| **教育制度** | 无公立学校 → 私立学校 → 公立学校 → 义务教育 |
| **宗教政策** | 国家宗教 → 宗教自由 → 完全分离 |

## 3. 社会博弈 (Domestic Power Dynamics)

### Victoria 3 美国内部势力互动

#### 利益集团博弈图谱
```
                    [废奴压力]
                        ↓
    ┌─────────────────────────────────────┐
    │                                     │
地主 ←──冲突──→ 知识分子/实业家              │
(南方)         (北方)                       │
    │            ↑                         │
    └────────────┴─────────────────────────┘
                        ↓
                    [内战触发]
```

#### 关键博弈机制
1. **废奴议程**: 知识分子+实业家推动 → 地主强烈反对 → 触发分裂主义
2. **关税之争**: 保护主义 vs 自由贸易 → 南北经济利益冲突
3. **移民政策**: 劳动力需求 vs 文化保守主义
4. **劳工运动**: 工业化后期 → 工会崛起 → 社会福利改革压力

#### 内战触发条件与后果
| 条件 | 触发阈值 | 结果 |
|------|----------|------|
| 废奴法案通过 | 地主激进度 > 80% | 南方分裂 |
| 奴隶制保留 | 知识分子激进度 > 80% | 北方激进废奴运动 |
| 军事斗争 | 战争胜败决定 | 胜者主导国家重建 |

#### 人口阶层 (Pop) 政治诉求
| Pop类型 | 主要诉求 | 激进度来源 |
|---------|----------|------------|
| **种植园主** | 维护奴隶制、低税收 | 废奴威胁 |
| **工厂主** | 自由市场、关税保护（分歧） | 劳工成本上升 |
| **工人** | 工作条件、工资、社会福利 | 生活水平下降 |
| **农民** | 土地权益、减税 | 土地税压力 |
| **奴隶** | 自由（废奴后） | 奴隶身份本身 |
| **移民** | 公民权、就业 | 文化排斥政策 |

### Civilization 6 社会模拟
- 人口不满 → 城市叛乱可能性
- 文化压力 → 外国文化渗透
- 宗教压力 → 宗教战争潜在触发

## 4. 外部政治 (Diplomacy & Expansion)

### Victoria 3 美国外交政策

#### 初期外交定位
| 维度 | 状态 | 说明 |
|------|------|------|
| **外交等级** | 区域大国 → 全球霸权 | 随工业化和军事发展 |
| **影响力** | 北美主导 | 门罗主义宣言 |
| **恶名** | 低 → 中（扩张） | 颦论扩张增加恶名 |
| **殖民能力** | 中等 | 西进运动、海外殖民 |

#### 特色外交机制
1. **门罗主义 (Monroe Doctrine)**: 
   - 声明美洲为美国势力范围
   - 阻止欧洲列强干预美洲事务
   - 日志条目触发外交事件

2. **天定命运 (Manifest Destiny)**:
   - 西进运动日志条目
   - 俄勒冈、加利福尼亚、德克萨斯扩张目标
   - 原住民驱逐/同化政策

3. **对外干涉**:
   - 墨西哥战争 → 吞并领土
   - 中美洲影响 → 巴拿马运河计划

#### 国际组织参与
| 组织 | 美国角色 | 时期 |
|------|----------|------|
| **国际联盟前身** | 观察员 | 19世纪后期 |
| **联合国** | 创始成员 | 二战后（Civ6） |

## 6. 开发者评论 (Design Philosophy Analysis)

### Victoria 3 美国设计解读

**设计意图**: 美国在 Victoria 3 中被设计为一个"从混乱到秩序"的典型案例。开发者意图模拟：
1. **民主制度的渐进性**: 从地产投票到普选权的漫长过程，体现民主扩展的历史复杂性
2. **利益集团冲突的核心性**: 地主 vs 实业家的博弈是美国政治的底层逻辑
3. **内战的必然性与偶然性**: 废奴议题不仅是道德问题，更是经济利益的终极碰撞

**玩家体验设计**: 美国玩家面临的核心挑战是"如何管理内战前的权力平衡"。过早推动废奴会触发分裂，过晚则被激进废奴派推动。这模拟了林肯等历史人物的困境。

**机制创新点**:
- 利益集团的"激进度"机制：量化政治冲突的爆发阈值
- 日志条目系统：将历史事件转化为可追踪的政治议程
- 分裂战争：展示政治分裂的军事后果

### Civilization 6 美国设计解读

**设计意图**: Civ6 的美国设计更偏向"文明特质"而非"政治模拟"。泰迪·罗斯福的"Rough Rider"能力体现美国扩张主义的历史记忆。

**与 V3 的对比**:
| 维度 | Victoria 3 | Civilization 6 |
|------|------------|----------------|
| **政治深度** | 利益集团博弈 | 政策卡选择 |
| **历史模拟** | 精确时间线 | 时代跃迁 |
| **社会冲突** | Pop激进度系统 | 城市叛乱概率 |
| **扩张机制** | 外交博弈+战争 | 文化炸弹+军事征服 |

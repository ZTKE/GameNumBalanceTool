## 1. 权力核心 (Who Rules? How to Change?)

### Victoria 3 中的英国权力结构

#### 初始政体设定 (1836年)
| 维度 | 状态 | 说明 |
|------|------|------|
| **治理原则** | 君主制 (Monarchy) | 维多利亚女王统治 |
| **权力分配** | 地产投票 (Landed Voting) | 贵族主导议会 |
| **继承机制** | 父系继承 | 王位世袭 |
| **国家宗教** | 国家宗教 (State Religion) | 英国国教会 |
| **议会类型** | 两院制 | 上院(贵族)+下院(选举) |

#### 核心利益集团 (Interest Groups)

| IG名称 | 初始权重 | 意识形态 | 政治立场 |
|--------|----------|----------|----------|
| **托利党 (Tories)** | 高 | 保守主义、传统主义 | 维护贵族特权 |
| **辉格党 (Whigs)** | 中高 | 自由主义、改革 | 推动议会改革 |
| **地主阶级 (Landowners)** | 极高 | 传统主义、保守 | 农业利益维护 |
| **实业家 (Industrialists)** | 高增长 | 自由市场、工业化 | 工业革命推动者 |
| **知识分子 (Intelligentsia)** | 中 | 自由主义、改革 | 议会改革核心 |
| **教会 (Devout)** | 中高 | 英国国教 | 维护宗教地位 |
| **工会 (Trade Unions)** | 极低→后期 | 工人权益 | 工业化后崛起 |
| **军队 (Armed Forces)** | 中 | 保守、帝国 | 维护殖民霸权 |
| **小资产阶级 (Petite Bourgeoisie)** | 中增长 | 民族主义、稳定 | 中产阶级 |

#### 权力更替的渐进路径
```
1836: 君主制 + 地产投票 (贵族主导)
  │
  ├──→ [1832改革法案] → 扩大选举权 (实际历史已发生)
  │
  ├──→ [1867改革法案] → 城市工人选举权
  │
  ├──→ [1884改革法案] → 农村工人选举权
  │
  └→ [1918选举法] → 女性选举权 + 普选权
```

**关键特点**: 英国改革是**渐进式**而非革命式，没有法国那样的剧烈政体变更。

## 2. 政治工具箱 (All Laws, Policies, Abilities)

### Victoria 3 英国法案矩阵

#### 权力结构法案
```
治理原则: 君主制 ✓ (维持至现代)
权力分配: 地产投票 → 户主投票 → 财富投票 → 普选权 (渐进)
继承机制: 父系继承 (稳定)
国家宗教: 国家宗教 → 宗教自由 → 完全分离 (渐进)
```

#### 英国特色法案路径
| 法案类型 | 英国演进路径 | 关键利益集团推动 |
|----------|--------------|------------------|
| **投票制度** | 地产→户主→财富→普选 | 辉格党+知识分子 |
| **经济体制** | 传统→农本→自由放任（快速） | 实业家主导 |
| **贸易政策** | 保护→自由贸易（1846废除谷物法） | 实业家推动 |
| **殖民制度** | 殖民扩张制度化 | 军队+实业家 |
| **劳工权利** | 无限制→逐步保护 | 工会后期推动 |
| **福利制度** | 无→社会保险→养老金 | 自由党改革 |

#### 关键历史法案
| 法案 | 年份 | 游戏效果 |
|------|------|----------|
| **1832改革法案** | 扩大选举权 | 下院权力增强 |
| **1846谷物法废除** | 自由贸易确立 | 实业家权重增加 |
| **1867改革法案** | 工人选举权 | 政治改革事件 |
| **1870教育法案** | 公立教育 | 知识分子支持 |
| **1884改革法案** | 农村选举权 | 普选接近 |
| **1911议会法案** | 上院权力削弱 | 下院主导 |
| **1918选举法** | 女性选举权 | 普选权完成 |

## 3. 社会博弈 (Domestic Power Dynamics)

### Victoria 3 英国内部势力互动

#### 利益集团博弈图谱
```
      [工业革命压力]
           ↓
┌──────────────────────┐
│                      │
托利党 ←──竞争──→ 辉格党
(保守)       (自由)
    │            │
    └───────────┴────→ 实业家崛起
           │
           ↓
    [地主权力衰退]
           │
           ↓
    [工会崛起(后期)]
```

#### 英国特有的政治博弈
1. **贵族 vs 工业资本**: 地主阶级逐步被实业家取代主导地位
2. **谷物法之争**: 保护主义 vs 自由贸易的标志性斗争
3. **议会改革**: 辉格党推动渐进民主化
4. **工人运动**: 工会后期的劳工权利争取

#### 渐进改革 vs 革命
| 特征 | 英国模式 | 法国模式对比 |
|------|----------|--------------|
| **改革速度** | 渐进 | 剧烈革命 |
| **政体稳定性** | 高（君主制延续） | 低（多次政体变更） |
| **权力转移** | 利益集团博弈 | 革命事件触发 |
| **社会冲突** | 低激进度 | 高激进度 |

## 6. 开发者评论 (Design Philosophy Analysis)

### Victoria 3 英国设计解读

**设计意图**: 英国是"渐进改革"的典范。开发者意图展示：
1. **贵族社会的转型**: 地主阶级如何被工业资本逐步取代
2. **议会民主的渐进性**: 无革命的民主扩展
3. **全球霸权的维护**: Pax Britannica的殖民秩序

**玩家体验**: 英国玩家的核心优势是**稳定性**，但挑战是：
- 如何管理渐进改革的节奏
- 如何维护全球殖民霸权
- 如何应对后期工人运动

**机制特点**:
- 利益集团权重的渐进变化
- 无剧烈革命事件链
- 殖民扩张的独特优势

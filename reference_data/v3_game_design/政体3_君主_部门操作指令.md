## 君主政体特殊机制说明

### 1. 王权转化机制
- **王权获取**: 血统继承、贵族支持、传统主义声量转化
- **王权消耗**: 分封爵位、特权授予、王室声明等
- **王权阈值**: 逐步解锁高级指令

### 2. 贵族阶层特殊地位
- **贵族军官**: 军队核心，但专业能力下降
- **贵族产业**: 经济核心，但市场效率下降
- **贵族特权**: 稳定核心，但底层激进化上升

### 3. 底层激进化管理
- **农奴制**: 最高激进化，需谨慎使用
- **贵族特权**: 激进化严重，需平衡
- **底层公民权缺失**: 激进化严重，需谨慎

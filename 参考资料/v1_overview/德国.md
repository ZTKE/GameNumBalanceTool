## 1. 权力核心 (Who Rules? How to Change?)

### Victoria 3 中的德国权力结构

#### 统一前：普鲁士 (Prussia) 初始设定 (1836年)
| 维度 | 状态 | 说明 |
|------|------|------|
| **治理原则** | 君主制 (Monarchy) | 普鲁士王国 |
| **权力分配** | 地产投票 (Landed Voting) | 容克贵族主导 |
| **继承机制** | 父系继承 | 霍亨索伦王朝 |
| **国家宗教** | 国家宗教 | 新教主导 |
| **军事体制** | 专业军队 | 军国主义传统 |

#### 核心利益集团 (Interest Groups) - 普鲁士

| IG名称 | 初始权重 | 意识形态 | 政治立场 |
|--------|----------|----------|----------|
| **容克贵族 (Junkers/Landowners)** | 极高 | 传统主义、军国主义 | 普鲁士核心势力 |
| **军队 (Armed Forces)** | 高 | 军国主义、威权主义 | 军事主导文化 |
| **实业家 (Industrialists)** | 中→高增长 | 自由主义、工业化 | 莱茵地区势力 |
| **知识分子 (Intelligentsia)** | 低→中 | 自由主义、民族主义 | 统一思想推动者 |
| **教会 (Devout)** | 中 | 新教保守 | 文化影响 |
| **小资产阶级 (Petite Bourgeoisie)** | 中增长 | 民族主义 | 统一战争支持者 |

#### 德国统一的多条路径
```

  │
  ├──→ [北德邦联] → [德意志帝国] → [第二次世界大战]
  │         │
```

## 2. 政治工具箱 (All Laws, Policies, Abilities)

### Victoria 3 德国法案矩阵

#### 权力结构法案
```
治理原则选项：
  ✓ 君主制 (帝国初始)
  ✓ 共和制 (威玛共和)
  ✓ 威权总统制 (纳粹前)
  ✓ 独裁制 (纳粹德国)

  地产投票 → 户主投票 → 普选权（威玛）
  独裁（纳粹）

继承机制：
  父系继承 → 选举总统 → 无继承（纳粹）
```

| 法案类型 | 德国演进路径 | 关键历史节点 |
|----------|--------------|--------------|

| **军事体制** | 专业军队→国家民兵→党卫军 | 军事国家化 |
| **经济体制** | 自由放任→国家干预→战争经济 | 战争导向 |

## 3. 社会博弈 (Domestic Power Dynamics)

### Victoria 3 德国内部势力互动

#### 利益集团博弈图谱
```
    [民族统一压力]
         ↓
┌─────────────────────┐
│                     │
容克 ←──主导──→ 实业家
(军事贵族)    (工业资本)
    │            │
    └──────────┬─┘
               │
               ↓
         [统一战争]
               │
               ↓
        [知识分子崛起]
               │
               ↓
      [威玛共和/纳粹崛起]
```

#### 德国特有的政治博弈
1. **容克 vs 工业资本**: 军国主义传统 vs 工业现代化
2. **统一民族主义**: 小资产阶级+知识分子的统一推动
3. **威玛危机**: 民主共和 vs 极权复辟
4. **纳粹崛起**: 小资产阶级激进化 + 军队支持

#### 威玛共和国的脆弱性
| 原因 | 游戏体现 |
|------|----------|
| **经济危机** | 通货膨胀事件 |
| **政治极化** | IG激进度上升 |
| **军队独立性** | Armed Forces权重过高 |

## 6. 开发者评论 (Design Philosophy Analysis)

### Victoria 3 德国设计解读

**设计意图**: 德国展示了"民族统一+极权崛起"的双重路径。开发者意图模拟：
1. **统一进程**: 三次战争的历史必然性
2. **威玛脆弱性**: 民主制度如何被极权取代
3. **军事传统**: 容克文化对德国的深远影响

**玩家体验**: 德国玩家的核心体验是：
- 统一战争的军事战略
- 威玛共和的政治危机管理
- 纳粹崛起的选择（历史路线 vs 假设路线）

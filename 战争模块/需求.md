# 游戏数值设计底层逻辑与武器数值规范全书

## 一、 全局基础：非线性阈值伤害计算
该公式是后续所有伤害计算的底层逻辑：
* **若 攻击 <= 防御**：基础伤害 = 攻击 * 0.1
* **若 攻击 > 防御**：基础伤害 = 防御 * 0.1 + (攻击 - 防御) * 0.4

## 二、 武器设计核心准则与约束

### 1. 距离变量体系
* **10阶段交战距离**：当变量标题记录10阶段时代表需设定10个数值。
* **距离区间**：0-0.5km, 0.5-1.5km, 1.5-4km, 4-8km, 8-20km, 20-40km, 40-80km, 80-120km, 120-240km, 240-480km。
* **填写格式**：必须使用 `=` 分隔（如 `1=2=8=6=0=0=0=0=0=0`），不要带 `*`。
* **差异化要求**：数值必须随距离产生变化，体现武器在不同射程下的优劣。设计要体现武器的优缺点差异化，比如有的武器是远距离压制强，有的武器是近距离压制强，有的武器是软攻强，有的武器是硬攻强类似这样差异化，不能发生一个武器可以完全被其他武器取代掉。

### 2. 设计规范与限制
* **不可修改项**：army.csv/air.csv/navy.csv中武器行数禁止修改，每列的武器种类、时代、成本、编制宽度、速度这 5 列数值禁止修改（这些是已经固定的锚定值，你可以参考）。你只需要考虑修改每行的其他变量数值。
* **差异化设计**：体现武器差异化（如软硬攻侧重、远近程压制力），确保武器具有不可替代性。
* **数值范围**：各种适应性、火控、各种命中、对地探测、潜艇隐身、隐身、拦截、成本损耗变量代表的数值是 0-1 之间的小数。
* **编制平衡锚定**：
    * 陆军团上限：2200 宽度
    * 空军大队上限：24 宽度
    * 海军舰队上限：60 宽度
    * 跨维度平衡：2200 陆军的对空面板应与 24 空军的对地面板保持平衡。
* **成本匹配**：武器面板应与其成本挂钩，例如坦克面板必须显著强于同比例成本下的步枪集群。

### 3. 能耗与经济锚定
数值以 **1小时** 为单位。燃料/核动力消耗需参考一个5000个人工厂产出：
* **燃料产出**：一战5000个人工厂 0.25/h | 二战5000个人工厂 0.83/h | 冷战5000个人工厂 2.92/h | 现代5000个人工厂 7.5/h。
* **核动力产出**：冷战5000个人工厂 0.17/h | 现代5000个人工厂 0.42/h。

### 4. 战斗逻辑备注
* **突破与防御**：突破代表进攻时防御，防御代表防御时防御。即便坦克，突破数值也应小于防御数值。
* **友伤**：大口径火炮在超近距离有友伤；步枪等武器友伤设为 0。
* **战斗节奏**：海战有一方一半以上船只失去战斗力（浮力值、结构值或组织度归零代表失去战斗力）则战役结束，海战每轮是舰队内每个船独立伤害结算敌方舰队某个随机船只。陆战和空战有一方组织度或血量0时则战役结束，陆战和空战是一个编制所有武器总面板和敌方编制总面板之间的伤害结算。海陆空二战时期战役持续时间尽量保持和游戏钢铁雄心4一致，例如二战中钢铁雄心4一个装甲师打一个步兵师需要多久损耗多少，我们这里就是一个装甲团打一个步兵团就是需要打多久损耗多少，其他时代结合现实等比例映射。


### 5. 陆军战斗距离设计
* 陆军开局从20-40km距离开始交战，进攻方推进度 = 进攻方速度/(进攻方速度+8) * 0.25 + (进攻方突破-防守方压制+进攻方压制/2)/(进攻方压制/2+进攻方突破+防守方压制) * 0.75 + [-0.2~0.3]之间随机数。进攻方推进度 > 0.2 则缩短前一个交战距离，进攻方推进度 < -0.3 则延长后退一个交战距离，陆战交战距离限制在0-40km内。
* 海军和空军从雷达探测到那一刻双方距离算间距开始交战，进攻方推进度 = (进攻方速度-防御方速度)/(进攻方速度+防御方速度) + [-0.2~0.3]之间随机数。进攻方推进度 > 0.2 则缩短前一个交战距离，进攻方推进度 < -0.3 则延长后退一个交战距离，海空交战距离没有限制。
* 海陆空每个部队在阶段1（0-0.5km）结算完成后自动重置到20-40km（阶段6）重新循环结算，直到一方组织度或血量归零。

### 6. 特殊武器备注
* 所有air.csv内武器防御数值填0即可，不需要设计防御变量。air.csv中仅侦察机和预警机有对地探测、雷达强度、雷达半径3个变量有具体数值。空战时战斗机等作战编队的对地探测值可以由战场外的侦察机或预警机编队直接提供（但不提供攻防数值）。
* 船只设计数值面板时仅考虑船只本身属性，不考虑其携带的飞机数值。

## 三、 综合每小时伤害判定逻辑

### 1. 陆军伤害判定
* **地对地**：
    1. **命中率** = min(1.0, max(0.05, 对地命中 + 火控 * max(0, 1 - 敌方电子干扰 / (1 + 电子抗性))))
    2. **敌方装甲率** = 敌方装甲类武器编制宽度总和 / 敌方部队总编制宽度。（装甲类武器判定：army.csv中"武器定位"编号=2的单位为装甲类武器，如坦克、步兵战车等。例：一个团内装甲单位宽度合计1000，总宽度2200，则装甲率=1000/2200≈45.5%）
    3. **有效攻击** = 敌方装甲率 * 己方对地硬攻 + (1 - 敌方装甲率) * 己方对地软攻
    4. **敌方有效防御** = 敌方堑壕防御 + (敌方处于防御状态 ? 敌方防御 : 敌方突破)
    5. **基础伤害** = 将“有效攻击”与“敌方有效防御”代入全局基础公式
    6. **击穿判定**：若己方对地装甲穿透 > 敌方装甲厚度，击穿效率 = 1；否则击穿效率 = max(0.1, 己方对地装甲穿透 / (敌方装甲厚度 + 1))
    7. **最终伤害** = 基础伤害 * 击穿效率 * 命中率 * 环境适应系数
    8. **环境适应系数计算**：
        * 环境适应性变量包括：海洋适应性、平原适应性、城市适应性、丘陵适应性、森林适应性、丛林适应性、山地适应性、沙漠适应性、沼泽适应性、可通行冰面湖泊适应性、酷暑适应性、炎热适应性、温和适应性、严寒适应性、极寒适应性、晴朗适应性、小雨适应性、暴雨适应性、小雪适应性、暴风雪适应性、沙尘暴适应性、大雾适应性。
        * **单环境**：部队环境适应系数 = Σ(武器数量 × 武器编制宽度 × 武器环境适应性) / 部队总编制宽度
        * **多环境叠加**（如暴雨+平原）：综合适应系数 = 所有环境适应性的平均值。例：暴雨适应性0.8，平原适应性0.9 → 综合 = (0.8 + 0.9) / 2 = 0.85；若有3个环境（暴雨0.8 + 平原0.9 + 夜间0.7）→ 综合 = (0.8 + 0.9 + 0.7) / 3 = 0.8
    9. **结算**：敌方血量 - 最终伤害 * 0.1；敌方组织度 - 最终伤害

* **地对空/海对空/空对空**：
    1. **命中逻辑**：基础导引 = 对空命中 + 火控 * (1 - 敌方干扰 / (1 + 电子抗性)) - 敌方拦截；（拦截变量主要是代表飞机对敌方火力的拦截率，但是超近距离交战贴脸时拦截变量也综合考虑一定程度上飞机闪避的逻辑）
    2. **命中率** = min(1.0, max(0.05, 基础导引))
    3. **伤害计算**：最终伤害 = 对空伤害 * 命中率。（空对空战斗没有击穿判定，直接使用"对空伤害"变量结算血量）
    4. **结算**：敌方血量 - 最终伤害 * 0.3；敌方组织度 - 最终伤害。

* **地对海**：
    1. **命中逻辑**：
        * 目标为军舰：基础导引=对海命中 + 火控 * (1 - 敌方电子干扰 / （1+电子抗性）) - 敌方拦截（拦截变量主要是代表船只对敌方火力的拦截率，但是超近距离交战贴脸时拦截变量也综合考虑一定程度上船只闪避的逻辑）
        * 目标为潜艇：基础导引=(对海命中 + 火控 * (1 - 敌方电子干扰 / （1+电子抗性）) - 敌方拦截) * 声呐强度 / (敌方潜艇隐身 + 1))。（陆军武器无声呐强度变量，所以陆军这里基础导引就是0，海军和空军则有声呐强度来推算基础导引）
    2. **命中率** = min(1.0, max(0.05, 基础导引))
    3. **伤害修正**
        * 目标军舰：有效攻击 = 对海伤害
        * 目标潜艇：有效攻击 = 对潜伤害
    4. **基础伤害** = 将“有效攻击”与“敌方防御”代入全局基础公式。
    5. **击穿判定**：若己方对海装甲穿透 > 敌方厚度，击穿效率 = 1；否则击穿效率 = max(0.1, 己方对海装甲穿透 / (敌方装甲厚度 + 1))
    6. **结算**：最终伤害 = 基础伤害 * 击穿效率 * 命中率。敌方结构值 - 最终伤害；敌方浮力值 - 最终伤害 * 0.3；敌方组织度 - 最终伤害。

### 2. 空军伤害判定
* **空对地/海对地**：
    1. **基础导引**：基础导引 = 对地命中 + 火控 * max(0, 1 - 敌方电子干扰 / (1 + 电子抗性))
    2. **命中率**：命中率 = min(1.0, max(0.05, 基础导引 * 对地探测))；
    3. **伤害逻辑**：同陆军地对地公式。空军使用"对地装甲穿透"变量进行击穿判定，敌方装甲厚度取地面单位的装甲厚度。
    4. **结算**：敌方血量 - 最终伤害 * 0.1；敌方组织度 - 最终伤害。

* **空对海**：
    1. **判定逻辑**：同地对海。
    2. **结算差异**：敌方结构值 - 最终伤害；敌方浮力值 - 最终伤害 * 0.5；敌方组织度 - 最终伤害。

### 3. 海军伤害判定 (海对海)
* **命中率**：同地对海。
* **伤害修正**：
    * 目标军舰：
        * 目标为轻甲船：轻炮对海伤害 = 轻炮对海伤害*1; 重炮对海伤害 = 重炮对海伤害*1.2; 导弹对海伤害 = 导弹对海伤害*1;
        * 目标为重甲船：轻炮对海伤害 = 轻炮对海伤害*0.8; 重炮对海伤害 = 重炮对海伤害*1; 导弹对海伤害 = 导弹对海伤害*0.9;
        * **轻甲/重甲判定**：navy.csv中"装甲类型"列，0=轻甲船（如驱逐舰、护卫舰），1=重甲船（如战列舰、重型巡洋舰）。
    * 目标潜艇：有效攻击 = 对潜伤害
* **击穿判定**：若己方对海装甲穿透 > 敌方厚度，击穿效率 = 1；否则击穿效率 = max(0.1, 己方对海装甲穿透 / (敌方装甲厚度 + 1))
* **结算**：
    目标非潜艇:
        * **浮力值损耗** = 击穿效率 * 命中率 * ((轻炮对海伤害*0.05 + 重炮对海伤害*0.4 + 鱼雷对海伤害*3 + 导弹对海伤害*0.2总和值)与“敌方防御”代入全局基础公式)
        * **结构值损耗** = 击穿效率 * 命中率 * ((轻炮对海伤害*1 + 重炮对海伤害*1 + 鱼雷对海伤害*0.3 + 导弹对海伤害*1.5总和值)与“敌方防御”代入全局基础公式)
    目标潜艇：
        * **浮力值损耗** = 击穿效率 * 命中率 * (对潜伤害与“敌方防御”代入全局基础公式)
        * **结构值损耗** = 击穿效率 * 命中率 * (对潜伤害与“敌方防御”代入全局基础公式)
* 海军舰队是每个船只独立开火敌方随机一个船只，也就是一个船的面板对抗敌方随机一个船只的面板。每轮双方所有船都会开火一次。

*海陆空每轮由进攻方发起攻击，防御方同时也会相同伤害公式反击。通过双方侦查值对比来决定本轮谁先开火，侦查值取当前交战距离对应阶段的"侦查（10阶段）"变量值。比如进攻方侦查值50，防守方侦查值30，则进攻方本轮50/(50+30)的概率先开火。
*然后血量每损失百分之1则所有面板属性下降百分之1。（先开火之后对手反击应该是扣过血之后的面板反击）


附加备注：
陆军突破是1个数值格式，海陆空防御都改成1个数值格式（即是1个值而不是10阶段格式，空军防御都写死为0，海军陆军有值）。
空军的雷达半径应该只填一个值，不应该是10阶段格式。空军和海军雷达强度，对地探测需要10阶段格式的数值，而不是一个值。
空军对潜伤害全部都是0不合理，比如海军直升机这些飞机要有对潜伤害。然后只有真正具备探测的到潜艇能力的空军才能有声呐强度数值。

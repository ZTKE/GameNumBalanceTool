# 游戏数值设计底层逻辑与武器数值规范全书

## 一、全局基础：非线性阈值伤害计算

该公式是后续所有伤害计算的底层逻辑：

* **若 攻击 <= 防御**：基础伤害 = 攻击 * 0.1
* **若 攻击 > 防御**：基础伤害 = 防御 * 0.1 + (攻击 - 防御) * 0.4

## 二、武器设计核心准则与约束

### 1. 距离变量体系

* **10阶段交战距离**：当变量标题记录10阶段时代表需设定10个数值。
* **距离区间**：0-0.5km, 0.5-1.5km, 1.5-4km, 4-8km, 8-20km, 20-40km, 40-80km, 80-120km, 120-240km, 240-480km。
* **填写格式**：必须使用 `=` 分隔（如 `1=2=8=6=0=0=0=0=0=0`），不要带 `*`。
* **差异化要求**：数值必须随距离产生变化，体现武器在不同射程下的优劣。设计要体现武器的优缺点差异化，比如有的武器是远距离压制强，有的武器是近距离压制强，有的武器是软攻强，有的武器是硬攻强类似这样差异化，不能发生一个武器可以完全被其他武器取代掉。

### 2. 设计规范与限制

* **不可修改项**：武器行数、武器种类、时代、成本、编制宽度、速度这 5 列数值禁止修改。
* **差异化设计**：体现武器差异化（如软硬攻侧重、远近程压制力），确保武器具有不可替代性。
* **数值范围**：适应性、火控、命中、对地探测、拦截等变量代表的数值是 0-1 之间的小数。
* **编制平衡锚定**：
    * 陆军团上限：2200 宽度
    * 空军大队上限：24 宽度
    * 海军舰队上限：60 宽度
    * 跨维度平衡：2200 陆军的对空面板应与 24 空军的对地面板保持平衡。
* **成本匹配**：武器面板应与其成本挂钩，例如坦克面板必须显著强于同比例成本下的步枪集群。

### 3. 能耗与经济锚定

数值以 **1小时** 为单位。燃料/核动力消耗需参考工厂产出：

* **燃料产出**：一战 0.25/h | 二战 0.83/h | 冷战 2.92/h | 现代 7.5/h。
* **核动力产出**：冷战 0.17/h | 现代 0.42/h。
* **限制**：单个武器的每小时消耗不允许超过 5000 人工厂的日产出燃料。

### 4. 战斗逻辑备注

* **突破与防御**：突破代表进攻时防御，防御代表防御时防御。即便坦克，突破数值也应小于防御数值。
* **友伤**：大口径火炮在超近距离有友伤；步枪等武器友伤设为 0。
* **战斗节奏**：海战一半以上船只失去战斗力（浮力值、结构值或组织度归零）则战役结束。满编主力舰队打满编主力舰队海战考虑游戏性不能1-2小时就过快结束战斗，数值设计可以设计持续交战适当延长一倍时间，时间依然要在合理范围内。

### 5. 陆军战斗距离设计

* 陆军开局从20-40km距离开始交战，进攻方推进度 = 进攻方速度/(进攻方速度+8) * 0.25 + (进攻方突破-防守方压制+进攻方压制/2)/(进攻方压制/2+进攻方突破+防守方压制) * 0.75 + [-0.2~0.3]之间随机数。进攻方推进度 > 0.2 则缩短前一个交战距离，进攻方推进度 < -0.3 则延长后退一个交战距离，陆战交战距离限制在0-40km内。
* 海军和空军从雷达探测到那一刻双方距离算间距开始交战，进攻方推进度 = (进攻方速度-防御方速度)/(进攻方速度+防御方速度) + [-0.2~0.3]之间随机数。进攻方推进度 > 0.2 则缩短前一个交战距离，进攻方推进度 < -0.3 则延长后退一个交战距离，海空交战距离没有限制。
* 海陆空每个部队都会有一个自己最期望的交战距离，当进攻部队到达这个距离后即时满足可以前进距离也会保持在原地距离

## 四、综合每小时伤害判定逻辑

### 1. 陆军伤害判定

#### 地对地

1. **命中率** = min(1.0, max(0.05, 对地命中 + 火控 * max(0, 1 - 敌方电子干扰 / (1 + 电子抗性))))
2. **敌方装甲率** = 敌方装甲类武器编制宽度总和 / 敌方部队总编制宽度
3. **有效攻击** = 敌方装甲率 * 己方对地硬攻 + (1 - 敌方装甲率) * 己方对地软攻
4. **敌方有效防御** = 敌方堑壕防御 + (敌方处于防御状态 ? 敌方防御 : 敌方突破)
5. **基础伤害** = 将"有效攻击"与"敌方有效防御"代入全局基础公式
6. **击穿判定**：若己方对地装甲穿透 > 敌方装甲厚度，击穿效率 = 1；否则击穿效率 = max(0.1, 己方对地装甲穿透 / (敌方装甲厚度 + 1))
7. **最终伤害** = 基础伤害 * 击穿效率 * 命中率 * 环境适应
8. **结算**：敌方血量 - 最终伤害 * 0.1；敌方组织度 - 最终伤害

#### 地对空/海对空/空对空

1. **命中逻辑**：基础导引 = 对空命中 + 火控 * (1 - 敌方干扰 / (1 + 电子抗性)) - 敌方拦截；（拦截变量主要是代表飞机对敌方火力的拦截率，但是超近距离交战贴脸时拦截变量也综合考虑一定程度上飞机闪避的逻辑）
2. **命中率** = min(1.0, max(0.05, 基础导引))
3. **结算**：最终伤害 = 对空伤害 * 击穿效率 * 命中率。敌方血量 - 最终伤害 * 0.3；敌方组织度 - 最终伤害。

#### 地对海

1. **命中逻辑**：
    * 目标为军舰：基础导引=对海命中 + 火控 * (1 - 敌方电子干扰 / （1+电子抗性）) - 敌方拦截（拦截变量主要是代表船只对敌方火力的拦截率，但是超近距离交战贴脸时拦截变量也综合考虑一定程度上船只闪避的逻辑）
    * 目标为潜艇：基础导引=(对海命中 + 火控 * (1 - 敌方电子干扰 / （1+电子抗性）) - 敌方拦截) * 声呐强度 / (敌方隐身 + 1))。（陆军武器无声呐强度变量，所以这里基础导引就是0）
2. **命中率** = min(1.0, max(0.05, 基础导引))
3. **伤害修正**
    * 目标军舰：有效攻击 = 对海伤害
    * 目标潜艇：有效攻击 = 对潜伤害
4. **基础伤害** = 将"有效攻击"与"敌方防御"代入全局基础公式。
5. **击穿判定**：若己方对海装甲穿透 > 敌方厚度，击穿效率 = 1；否则击穿效率 = max(0.1, 己方对海装甲穿透 / (敌方装甲厚度 + 1))
6. **结算**：最终伤害 = 基础伤害 * 击穿效率 * 命中率。敌方结构值 - 最终伤害；敌方浮力值 - 最终伤害 * 0.3；敌方组织度 - 最终伤害。

### 2. 空军伤害判定

#### 空对地/海对地

1. **命中率**：同地块有己方部队则 = min(1.0, max(0.05, 基础导引 * 对地探测))；同地块无己方部队= min(1.0, max(0.05, 基础导引 * 对地探测 * 对地探测))；
2. **伤害逻辑**：同陆军地对地公式（有效防御、击穿判定等）。
3. **结算**：敌方血量 - 最终伤害 * 0.1；敌方组织度 - 最终伤害。

#### 空对海

1. **判定逻辑**：同地对海。
2. **结算差异**：敌方结构值 - 最终伤害；敌方浮力值 - 最终伤害 * 0.5；敌方组织度 - 最终伤害。

### 3. 海军伤害判定 (海对海)

* **命中率**：同地对海。
* **伤害修正**：
    * 目标军舰：
        * 目标为轻甲船：轻炮对海伤害 = 轻炮对海伤害*1; 重炮对海伤害 = 重炮对海伤害*1.2; 导弹对海伤害 = 导弹对海伤害*1;
        * 目标为重甲船：轻炮对海伤害 = 轻炮对海伤害*0.8; 重炮对海伤害 = 重炮对海伤害*1; 导弹对海伤害 = 导弹对海伤害*0.9;
    * 目标潜艇：有效攻击 = 对潜伤害
* **击穿判定**：若己方对海装甲穿透 > 敌方厚度，击穿效率 = 1；否则击穿效率 = max(0.1, 己方对海装甲穿透 / (敌方装甲厚度 + 1))
* **最终伤害** = 有效攻击 *
* **结算 (非潜艇目标)**：
    * **浮力值损耗** = 击穿效率 * 命中率 * (轻炮对海伤害*0.05 + 重炮对海伤害*0.4 + 鱼雷对海伤害*3 + 导弹对海伤害*0.2)
    * **结构值损耗** = 击穿效率 * 命中率 * (轻炮对海伤害*1 + 重炮对海伤害*1 + 鱼雷对海伤害*0.3 + 导弹对海伤害*1.5)
* **结算 (潜艇目标)**：
    * **浮力值损耗** = 击穿效率 * 命中率 * 对潜伤害
    * **结构值损耗** = 击穿效率 * 命中率 * 对潜伤害

## 六、一二战阵位系统

### 舰队阵型的优势程度

**接战状态判定**：通过双方舰队先手值的差值决定接战状态

**先手值** = 舰队速度 * 舰队射程/2 * √舰队侦察

| 先手值差距 | 接战状态 | 说明 |
|-----------|---------|------|
| > 35% | 伏击战 | 先手值大的为进攻方，小的为防守方 |
| 15%~35% | 对攻战 | 双方均为进攻方 |
| ≤ 15% | 遭遇战 | 双方均为防守方 |

**进攻方阵位** = 进攻方进攻权重 - 进攻方防守权重

**进攻方进攻权重** = 进攻方4~6级舰艇主武器数量总和（单个N联装主炮算N个）+ 进攻方所有舰艇平均速度（KM/H）+ 进攻方0级舰艇鱼雷发射管数量 + 舰载机数量/3

**进攻方防守权重** = 进攻方4~8级舰艇结构值总和（单位万吨）* 3 + 进攻方0~3级舰艇结构值总和（单位万吨）* 0.5

**防守方阵位** = 防守方反击权重 + 防守方防守权重

**防守方反击权重** = 防守方4~6级舰艇主武器数量总和（单个N联装主炮算N个）/ 3 + 防守方1~3级舰艇数量/2 + 舰载机数量/6

**防守方防守权重** = 防守方4~8级舰艇结构值总和（单位万吨）+ 防守方1~3级舰艇数量/1.5 + 舰载机数量/5 + 舰队平均速度

## 七、AI指挥思路

### 优势局

1. 己方突出部设为弱势一侧。
2. 主力集中中间配合另一侧部队多打少尝试包围敌人突出部，然后己方突出部弱势一侧可以缓缓撤退防御。
3. 突破敌人突出部后，一部分装甲力量突击己方弱的被敌人打穿那一侧的后方，然后中部部队回防被打穿薄弱那一侧。

### 均势局

先不主动发起攻击，兵力集中中间随时支援可能被敌人突破的一侧。己方防线能维持且发现敌人突出部后侧薄弱时，中间主力尝试包围敌人突出部。

### 劣势局

永远不主动发起攻击，兵力集中中间随时支援可能被敌人突破的一侧。

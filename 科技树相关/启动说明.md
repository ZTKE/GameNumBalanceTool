## 🐛 常见问题

### Q: 为什么打开旧HTML文件没有数据？

**A:** 旧HTML文件（`科技树可视化.html`）使用内嵌数据，不加载外部JSON。请使用新版本。

### Q: 为什么新HTML文件加载失败？

**A:** 可能原因：
1. JSON文件有注释（浏览器无法解析） → 已修复，使用 `_clean.json`
2. 文件不在同一目录 → 请确认文件位置
3. 浏览器缓存问题 → 清除缓存或强制刷新（Ctrl+F5）

### Q: 如何验证数据正确？

**A:** 运行Python分析工具：
```bash
python analyze_tech_tree.py
```

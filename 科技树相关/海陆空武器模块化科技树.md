# 二战至现代战争游戏：海陆空武器模块化科技树设计

---

## 一、时代划分

| 时代代号 | 时代名称 | 时间范围 | 技术特征 |
|---------|---------|---------|---------|
| `ww2` | 二战 | 1939-1945 | 螺旋桨飞机、大舰巨炮、轻/中/重型坦克分野 |
| `coldWarEarly` | 冷战前期 | 1945-1960 | 喷气机初期、核潜艇诞生、主战坦克概念出现 |
| `coldWarLate` | 冷战后期 | 1960-1990 | 导弹化、信息化初步、三代坦克 |
| `modern` | 现代 | 1990+ | 隐身技术、垂直发射系统、无人机、信息化作战 |

---

## 二、节点类型定义

| 类型代号 | 类型名称 | 说明 |
|---------|---------|------|
| `Unit` | 完整武器单位 | 如"M4谢尔曼坦克"、"F-15战斗机" |
| `Module` | 关键技术组件 | 如"大功率柴油机"、"相控阵雷达" |
| `Upgrade` | 升级套件 | 如"复合装甲套件"、"隐身涂层" |

---

## 三、非线性逻辑示例

### 一对多（一个基础科技开启多个后续科技）

```
大功率柴油机 [land_high_power_diesel]
    ├── 重型坦克I [land_heavy_tank_1]
    ├── 自行火炮I [land_sp_gun_1]
    └── 履带式步兵战车 [land_ifv]
```

### 多对一（必须解锁多个前提科技才能解锁某个高级模块）

```
主战坦克III [land_mb_tank_3]
    ├── 主战坦克II [land_mb_tank_2]
    ├── 滑膛炮技术 [land_smoothbore_gun]
    └── 稳像式火控 [land_stabilized_fc]
```

```
五代隐身战斗机 [air_gen5_fighter]
    ├── 四代战斗机 [air_gen4_fighter]
    ├── 隐身构型 [air_stealth_design]
    └── AESA雷达 [air_aesa]
```

```
神盾驱逐舰 [sea_ddg_2]
    ├── 导弹驱逐舰 [sea_ddg_1]
    ├── 相控阵雷达 [sea_phase_array_radar]
    └── 垂直发射系统 [sea_vls]
```

## 四、陆军科技树关键路径

### 4.1 基础模块层

```
基础内燃机 → 高功率汽油机 / 大功率柴油机
线膛炮技术 → 滑膛炮技术
均质装甲 → 复合装甲 → 间隙装甲
基础火控 → 稳像式火控 → 数字化火控
```

### 4.2 坦克演进线

```
轻型坦克I (WW2)
    └── 中型坦克I (WW2)
            ├── 中型坦克II (WW2) ──→ 主战坦克I (CW_E) ──→ 主战坦克II (CW_L) ──→ 主战坦克III (CW_L) ──→ 主战坦克IV (MOD)
            │                         ↑                      ↑                       ↑
            │                         │                      │                       │
            └── 重型坦克I (WW2) ──→ 重型坦克II (WW2) ───────┘──────────────────────┘
```

### 4.3 步兵支援线

```
栓动步枪 → 半自动步枪 → 突击步枪
步兵炮 → 无后坐力炮 → 单兵反坦克导弹
反坦克步枪 → 反坦克炮 → 反坦克歼击车
```

### 4.4 炮兵演进线

```
野战火炮
    ├── 重型火炮 (军属)
    ├── 自行火炮I → 自行火炮II
    └── 火箭炮 → 火箭炮II (师属)
防空炮 → 防空自行火炮 / 便携式防空导弹
```

### 4.5 新型作战线

```
轮式装甲车 → 履带式步兵战车
陆军直升机 → 武装直升机
             ↘ 无人机系统 (师属)
地对地导弹 (军属) / 地对空导弹 (军属)
```

## 五、海军科技树关键路径

### 5.1 基础系统层

```
声纳系统: 基础声纳 → 阵列式声纳 → 拖曳式声纳 → 主动式声纳
雷达系统: 机械扫描雷达 → 三坐标雷达 → 相控阵雷达
火控系统: 机械弹道计算机 → 脉冲火控雷达 → 电控火控雷达 → 数字火控雷达
电子战:   噪声诱饵系统 → 大功率压制系统
```

### 5.2 舰体演进线

```
潜艇演进:
潜艇 → 常规潜艇II → 帷规潜艇III
                        ↘ 核动力推进 ──→ 攻击核潜艇 ──→ 战略核潜艇

水面舰演进:
护卫舰I → 驱逐舰I → 导弹驱逐舰 ──→ 祖盾驱逐舰
    │         │         ↑              ↑
    │         │         │              │
    ├── 轻型巡洋舰 → 重型巡洋舰 ──→ 导弹巡洋舰 ──┘
    │         │
    │         ├── 战列巡洋舰
    │         └── 战列舰 → 战列舰II (终结)
```

### 5.3 航母演进线

```
轻型航空母舰I → 重型航空母舰I → 重型航空母舰II → 重型航空母舰III → 现代航空母舰
        │           ↑              ↑               ↑               ↑
        │           │              │               │               │
        └── 滑跃式甲板 ──→ 蒸汽弹射器 ──┘──────────┘──────────────┘
                                        │
                                        └── 电磁弹射器 ──────────────┘

两栖舰: 两栖攻击舰I → 两栖攻击舰II → 两栖登陆舰
```

### 5.4 导弹化转型

```
鱼雷: 基础鱼雷 → 先进鱼雷 → 磁性鱼雷 → 自导鱼雷
反舰导弹: 亚音速反舰导弹 → 超音速反舰导弹
防空导弹: 防空导弹 → 小型防空导弹 → 垂直发射系统 (多对一汇聚)
近防系统: 密集阵近防炮 → 多目标近防炮
```

## 六、空军科技树关键路径

### 6.1 基础技术层

```
发动机: 螺旋桨发动机 → 喷气发动机 → 加力燃烧室 → 涡扇发动机
航电:   基础航电 → 雷达火控 → 多功能雷达 → AESA雷达
机体:   基础机体 → 后掠翼设计 → 变后掠翼 → 隐身构型 → 复合材料机体
```

### 6.2 战斗机演进线

```
轻型战斗机 (WW2)
    ├── 重型战斗机 (WW2)
    │       └── 对海轰炸机
    └── 早期喷气机 (WW2)
            └── 二代喷气战斗机 (CW_E)
                    └── 三代多用途战斗机 (CW_L)
                            └── 四代战斗机 (CW_L)
                                    └── 五代隐身战斗机 (MOD)
                                        ↑ ↑ ↑
                                        │ │ │
                            隐身构型 ──┘ │ │
                            AESA雷达 ────┘ │
                            复合材料机体 ──┘

对海战斗机 (WW2) ← 轻型战斗机
```

### 6.3 轰炸机演进线

```
战术轰炸机I → 战术轰炸机II → 战术轰炸机III
    │
    └── 战略轰炸机I → 战略轰炸机II → 战略轰炸机III
    │                   ↑
    │                   │
    └── 强击机I → 强击机II
```

### 6.4 支援飞机线

```
侦察机 → 预警机 → 电子战飞机
运输机I → 运输机II → 空中加油机
海军直升机 (用于航母和两栖攻击舰)
```

# 经济系统设计文档

## 基础概念

### GOLD（核心锚点）
广义上的黄金，其实不是真的黄金，而是核心锚点。代表国家所有可兑换为国际等价物的财富总和（含虚拟逻辑黄金储备、工业产值、资源折算）。直观决定了一个国家的富裕程度。

### 流通货币 (Currency)
国家发行的纸币总额。包括居民池、企业池、国库池里的所有钞票总数。

### 名义财富
M pool玩家或 AI 在 UI 上直接看到的钞票数字。

### 实际购买力/货币实际价值 (Real)
每一单位货币对应的 G 含量。反映了"钱值不值钱"。

---

## 核心公式

```
货币实际购买力 = 国家GOLD总和 / 流通货币总和
```

**说明：**
- 如果国家 GOLD 总量不变，不断发行货币，就会导致购买力下降 → **通胀**
- 如果货币不变，但是国际交易导致 GOLD 减少同样也会导致 **通胀**

---

## 国际贸易逻辑

如果 A 国买了 B 国的商品，直接使用 A 国对应资金池里的 GOLD 与 B 国进行贸易。

**示例：**
- A 国买了 B 国 1 万单位小麦
- A 国对应资金池 -10000 GOLD
- B 国对应资金池 +10000 GOLD
- 资金池里面的数值直接使用 GOLD 即可
- 给玩家显示的 UI 可以通过实际购买力计算对应的货币

---

## 贸易顺差与贸易逆差

| 类型   | 条件                           | 货币影响                    |
| ---- | ---------------------------- | ----------------------- |
| 贸易顺差 | 回合结算 A 国获得 GOLD > A 国支出 GOLD | 货币总量不变 → **通缩**         |
| 贸易逆差 | 回合结算 A 国获得 GOLD < A 国支出 GOLD | 货币总量不变 → **通胀**（国家越来越穷） |

---

## 经济循环示意

```
            ┌─────────────────────────────────────┐
            │           国际贸易市场               │
            └─────────────────────────────────────┘
                          ↑ GOLD流入
                          ↓ GOLD流出
    ┌─────────────────────────────────────────────────┐
    │                    A国                          │
    │  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
    │  │ 居民池  │  │ 企业池  │  │ 国库池  │        │
    │  └─────────┘  └─────────┘  └─────────┘        │
    │       ↓            ↓            ↓             │
    │            流通货币总和                          │
    │                  ↓                              │
    │       实际购买力 = GOLD总量 / 货币总量           │
    └─────────────────────────────────────────────────┘
```

---

## 设计要点

1. **GOLD 是真实财富**：国际贸易直接转移 GOLD，不是转移纸币
2. **货币只是面子**：玩家看到的是名义货币，实际价值由 GOLD 决定
3. **通胀/通缩机制**：通过 GOLD 与货币的比例动态调整
4. **国家财富健康度**：GOLD 持续减少 = 国家变穷 = 通胀风险
